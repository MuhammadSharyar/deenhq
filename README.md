# DeenHQ 🌙

DeenHQ is a 100% client-side, privacy-first Islamic utility Progressive Web App (PWA). It is designed to be hosted entirely as static assets with zero backend architecture, ensuring that user data never leaves their device. 

## ✨ Features
- **Privacy First:** No databases, no third-party tracking, and no ads.
- **100% Offline Capable:** As a PWA, all core assets (including the complete Uthmani Quran dataset) are cached locally.
- **Prayer Times:** Dynamic, offline-first prayer calculations powered by the `adhan` library with a seamless midnight-rollover countdown.
- **Local Tracker:** A digital tasbih with haptic feedback and a daily habit checklist that automatically resets at midnight.
- **Quran Reader:** A beautifully typeset, distraction-free reading interface featuring the `Amiri Quran` font and local bookmarking.

## 🛠 Tech Stack
- **Framework:** React 19 + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide React
- **Hosting:** 100% Static (Vercel / Cloudflare Pages)

## 🚀 Running Locally
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
4. Build for production: `npm run build`
